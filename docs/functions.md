## FUNCTIONS AND RETURN TYPES

### db.create()
Description: Create A Database<br>
Example: <br>
```js
db.create("users");
```
<br>
Return Type: Boolean<true/false>

### db.delete()
Description: Delete A Database<br>
Example: <br>
```js
db.delete("users");
```
<br>
Return Type: Boolean<true/false>
<br>
